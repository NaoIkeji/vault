# Vault  
ローカル専用・プライバシー最優先の家計簿アプリ

Vault は、すべてのデータをユーザーの PC 内に安全に保存する、  
**ローカル専用・プライバシー最優先の家計簿アプリ**です。

クラウドにデータを送らず、外部サービスと連携しないため、  
個人情報や家計データが外部に漏れる心配がありません。

---

## ✨ 特徴

- **完全ローカル保存**  
  すべてのデータは PC 内のフォルダに保存されます。

- **プライバシー最優先設計**  
  外部 API やクラウドサービスを一切使用しません。

- **シンプルで使いやすい**  
  最初は CLI（コマンドライン）から始め、徐々に UI を追加予定。

- **分析に強い**  
  Python を使って、支出の傾向や月ごとの変化を可視化できます。

---

## 📁 プロジェクト構成（予定）
vault/ ├── main.py       # アプリのエントリーポイント 
├── data/                # 家計簿データ（ローカル保存） 
├── venv/                # 仮想環境 
└── README.md            # このファイル



---

## 🚀 使い方

### 1. 仮想環境を有効化
.\venv\Scripts\Activate.ps1



### 2. アプリを実行
python main.py
